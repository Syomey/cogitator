# Cogitator

Трекер боевых действий для настольных ролевых игр (D&D и не только) — веб-приложение, разворачивается как **Telegram Mini App** через GitHub Pages.

- **Источник:** `index_moba.html` в основном проекте.
- **Деплой:** этот файл копируется сюда как `index.html` и публикуется через GitHub Pages.
- **URL приложения:** `https://<логин>.github.io/cogitator/`

## Обновление
При изменениях в основном `index_moba.html` — перекопировать его в `cogitator/index.html`, закоммитить и запушить. GitHub Pages подхватит автоматически.

## Telegram Mini App
После публикации привязать URL страницы к боту в @BotFather (Menu Button / Mini App).
